Nested Buttonbar Creator 2.00
**************************************

Verwendung:
=====
Dieses Programm erzeugt bis zu 20 Buttonbardateien für Total Commander.
Jede Buttonbar wird mit einer vordefinierten Anzahl Buttons zum Wechsel von einer zur anderen Buttonbar erzeugt.
Die Nummer der aktiven Buttonbar wird gedrückt und zur einfachen Erkennung der eben verwendeten Buttonbar wird diese rot eingefärbt .

Wie es funktioniert:
===========
Das Programm wird Buttonbardateien in den folgenden drei Schritten erzeugen:
Schritt 1: hier legen Sie fest, wieviele Buttonbars Sie brauchen (Minimum 2 und Maximum 20).
Schritt 2: hier legen Sie fest, wo die Buttonbardateien gespeichert werden.
Schritt 3: in diesem Schritt können Sie eine bereits existierende Buttonbardatei importieren und deren Inhalt der von Ihnen gewählten Buttonnummer zuordnen.

Wenn Sie auf Erzeugen drücken, werden die Buttonbardateien in dem in Schritt 2 festgelegten Ordner erzeugt und eine Meldung wird Sie zum Übernehmen auffordern.

Wichtige Anmerkungen:
============

  • Es wird empfohlen, den Ordner des "Nested bar creator" innerhalb Ihres Total Commander Installationsverzeichnisses anzulegen. In diesem Fall wird die Umgebungsvariable %Commander_Path% verwendet, um auf die Zahlenicons und Bardateien zu verweisen.
  • Es wird empfohlen, die Bardateien im Standardordner zu speichern ("Nested bar creator\bars") oder in irgend einem anderen Ordner innerhalb Ihres Total Commander Installationsverzeichnisses.
  • Sie können auch andere Icons als die mit dem Programm mitgelieferten verwenden. Sie sollten Ihre Icons allerdings im Standardordner ("Nested bar creator\icons") ablegen. Ebenso sollten Sie die Standardnamen für die Icondateien benutzen ("Dn_x.ico", "Up_x.ico").
