Nested Button Bar Creator 2.00
**************************************

Назначение:
===========
Эта программа создаёт файлы панели инструментов Total Commander (от 2 до 20).
Каждая панель будет начинаться с предопределённых нумерованных кнопок, которые вы сможете использовать для переключения между панелями.
Кнопка активной панели отображается нажатой, а её номер - красным, что позволяет легко узнать, какая из панелей инструментов сейчас текущая.

Как использовать:
=================
Программа создаёт файлы для панелей инструментов в пошаговой последовательности:
Шаг 1: здесь вы определяете, сколько панелей инструментов вам необходимо (от 2 до 20).
Шаг 2: здесь вы определяете, где сохранять файлы с данными о панелях инструментов.
Шаг 3: здесь вы можете импортировать содержимое уже имеющегося файла панели инструментов в панель с выбранным вами номером.

Когда вы нажмёте кнопку Создать, файлы с данными о панелях инструментов будут созданы в папке, заданной вами в шаге 2. Будет также выведено пояснение, как далее применять созданные файлы.

Важные примечания:
==================

  • Рекомендуется размещать папку программы внутри каталога, куда установлен Total Commander, что позволит использовать переменную окружения %Commander_Path% в ссылках на значки с цифрами и на *.bar-файлы.
  • Рекомендуется сохранять *.bar-файлы в заданной по умолчанию папке ("Nested bar creator\bars") или в иной папке внутри каталога, куда установлен ваш Total Commander.
  • Вы можете использовать другие значки вместо поставляемых с программой, но должны сохранять ваши значки внутри заданной по умолчанию папки ("Nested bar creator\icons"). Также вы должны использовать заданное по умолчанию имя файла значка ("Dn_x.ico" и "Up_x.ico") для, соответственно, обычных и нажатых кнопок.
